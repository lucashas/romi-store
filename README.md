### 🛠️ NOTA SOBRE EL ERROR DE BUILDPACK
Si el error persiste en la consola de Firebase, es porque falta el archivo `package-lock.json`. Ejecuta este último comando en tu terminal para estar 100% seguro:

```bash
npm install && git add package-lock.json && git commit -m "Add lockfile" && git push
```
